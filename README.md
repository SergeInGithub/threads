## ℹ️ Description

A Next.js Instagram's Threads app clone but more powerful with more features the original app lacks.

- Next.js
- React
- Typescript
- Tailwindcss

## 📦 Installation

```bash
pnpm install
```

## 🚀 Running the app

```bash
# development
pnpm dev

# build
pnpm build

```
